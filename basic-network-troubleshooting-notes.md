# Basic Network Troubleshooting Notes

## Common Steps
1. Confirm the device has power
2. Check cable connections
3. Verify Wi-Fi or Ethernet connection
4. Confirm IP configuration
5. Restart the device
6. Test connectivity with ping
7. Check whether the issue affects one user or multiple users

## Common Problems
- Incorrect IP settings
- Loose cables
- DNS issues
- Router or switch connectivity issues
- Disabled network adapter

## Documentation
Always record:
- What the issue was
- When it started
- What steps were taken
- Whether the problem was resolved
