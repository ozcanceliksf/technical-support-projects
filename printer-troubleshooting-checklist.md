# Printer Troubleshooting Checklist

## Basic Checks
- Confirm the printer has power
- Check cable or network connection
- Make sure paper is loaded
- Check toner or ink status
- Restart printer and computer

## System Checks
- Confirm printer is set as default
- Check printer queue for stuck jobs
- Reinstall or update printer driver
- Verify network printer IP address
- Print a test page

## User Support Notes
- Confirm the exact error message
- Ask when the issue started
- Check whether other users can print
- Document the steps taken
